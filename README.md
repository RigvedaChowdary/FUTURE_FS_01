


# 🚀 FUTURE_FS_01 - Personal Portfolio Website

A modern, responsive personal portfolio website designed to showcase my skills, projects, leadership experience, and journey as a developer.

## 👨‍💻 About the Project

This portfolio was built as part of my Full Stack Web Development Internship at Future Interns.

The website serves as my digital portfolio, highlighting my technical skills, projects, achievements, leadership experience, and ways to connect with me. The goal was to create a clean, professional, and responsive platform that reflects both my technical and personal growth.

---

## 🌟 Features

* Fully responsive design (Mobile, Tablet & Desktop)
* Modern and professional UI
* Smooth scrolling navigation
* Interactive project showcase section
* Skills and technology stack display
* Leadership & experience highlights
* Contact section with social links
* Responsive cards and hover effects
* Clean and user-friendly layout
